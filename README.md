# Topmost Manager ✨

A tiny, fast, and beautiful Windows utility that lets you **force any window to stay always on top** with just one click.

Perfect for:
- Keeping chat windows visible while gaming
- Watching YouTube/tutorials while working
- Monitoring tools, calculators, or notes on top of everything

No installation required — just run the `.exe`!

![Screenshot](https://cdn.discordapp.com/attachments/1425449859741122690/1440339806973333514/image.png?ex=691dcc7a&is=691c7afa&hm=55e0a4fc4c2abed1b74cae51d0bbc241f18ff7e0e64fe42b3f10aa5f770e2b4e&)  

### Features
- Lists all currently running windows
- One-click **Make Topmost** or **Remove Topmost**
- Always stays on top itself (so you can always find it)
- Super lightweight (< 100 KB)
- No background processes or services
- Works with Chrome, Firefox, games, Notepad, Discord, Spotify — everything!

### How to Use
1. Download `TopmostManager.exe` from the [Releases](https://github.com/FlamesIsCool/Topmost-Manager/releases) page
2. Run it (double-click)
3. Select any window from the dropdown
4. Click **Make Topmost** → done!

> **Pro tip**: Right-click → **Run as administrator** to control apps that are running as admin (like some games or tools).
